## How to run

Create a virtual env:

```bash
$ mkvirtualenv --python=python3.6 sshooter
```

If you alredy have the enviroment:

```bash
$ workon sshooter
```

Install all dependencies: 

```bash
$ pip install -r requirements.txt
```

Run:

```bash
$ python src/main.py
```
